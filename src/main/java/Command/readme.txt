Можно представить данные паттер как пульты дистанционного управления ( инвокеры )
команды, которые они могут слать
и приемники - объекты, принимающие команды. Телевизор, умный светильник, стиралка и т д

У нас есть пульты
В них определены команды,  через интерфейс, а также методы, вызывающие эти команды
Конкретные команды-классы содержат приемник, с которым они работают. Они вызывают соответвующий метод \ методы у приемника

Пульты могут быть разными ( с разными наборами команд )
и приемники могут быть разные
